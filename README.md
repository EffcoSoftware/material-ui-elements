# material-ui-elements
Component library for common use cases. Based on material-ui & material-ui@next (as of 13 Jun 2017). Currently exporting the following components:

-  MuiTextfield
-  MuiSelectfield
-  MuiTable
-  MuiForm
-  CrudButtons
-  Fab

# Install
```
npm i -S material-ui-elements
```
